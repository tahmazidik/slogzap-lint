# slogzap-lint

Линтер для Go, совместимый с `golangci-lint`, который анализирует лог-записи и проверяет лог-сообщения на соответствие правилам.

Поддерживаемые логгеры:
- `log/slog`
- `go.uber.org/zap`

## Проверяемые правила

Линтер проверяет строковый литерал первого аргумента лог-вызова:

1. Сообщение начинается со строчной буквы
2. Сообщение только на английском (кириллица запрещена)
3. Сообщение не содержит спецсимволов/эмодзи
4. Сообщение не содержит потенциально чувствительных данных (по ключевым словам)

Правила соответствуют требованиям тестового задания.

## Быстрый старт

Сборка и запуск через `singlechecker`:

```bash
go test ./...
go run ./cmd/slogzaplint ./...

> Примечание: `examples/` содержит намеренно некорректные сообщения для демонстрации,
> поэтому запуск линтера на `./...` покажет диагностики и завершится с ненулевым кодом.

Проверка только исходного кода линтера (без examples):
